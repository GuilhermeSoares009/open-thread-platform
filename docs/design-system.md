# Design System (Initial)

This is a structural baseline and will be refined after Figma extraction.

## Tokens
- Colors: neutral base, primary accent, success, warning, danger
- Typography: one display family + one body family
- Spacing: 4px grid
- Radius: 6/10/16
- Shadow: soft elevation for cards

## Components
- Button (primary, secondary, ghost)
- Card (post, comment, community)
- Badge (tags, counts)
- Input, TextArea
- Avatar
- EmptyState, Loading

## States
- Default, hover, focus, disabled
- Loading, empty, error

## Accessibility
- Focus visible, adequate contrast, semantic headings
